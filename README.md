# 📌 Task Management Web Application

A simple and responsive Task Management Web Application built using HTML, CSS, and JavaScript. This project helps users organize and track their daily tasks efficiently.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Data persistence using Local Storage
- Clean and user-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
